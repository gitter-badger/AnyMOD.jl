Mapping

!!! Set a coarse temporal resolution for debugging Creating and especially solving a model is much faster if you decrease the temporal resolution of dispatch. Therefore, it is advisable to first test new models at less detailed temporal resolution. This will help you to spot and fix mistakes or unintended effects more efficiently.

Optional mappings

  • carrier_equality: optionen: ("no","yes"), default: "no"
